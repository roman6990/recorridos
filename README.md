# recorridos
scripts pyhthon para la revision de los segmentos
se requiere tener instalado:
* tesseract-ocr
* python

se debera descargar los archvios y mantenerlos dentro de una carpeta,
dentro de esa carpeta crear una carpeta que se llame fotos, en la cual se vaciaran las fotos a analizar

la app principal importara e instalara las librerias necesarias
anexo liga del tesseract:

https://github.com/tesseract-ocr/tesseract
